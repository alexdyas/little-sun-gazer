# Little Sun Gazer

A miniature device depicting the relative position of the Sun and moon.

![pic](docs/pic.jpg)

### Hardware
* Raspberry Pi Pico
* Precision RTC Module (DS3231)
* Waveshare e-Paper 3.7
* Pimoroni LiPo SHIM
* Li-Po 2000mAh 103450
